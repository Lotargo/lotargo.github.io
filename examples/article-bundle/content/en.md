# Example Article

This is the minimal Markdown source for a new publication. The first paragraph automatically becomes the article lead.

## First section

Write the idea, evidence, and structure. The renderer creates the bilingual HTML page, metadata bar, typography, navigation shell, and image components from the source and `article.json`.

> Markdown remains the readable semantic source, while HTML becomes a generated publication artifact.

## Supported content

The default renderer supports headings, paragraphs, emphasis, links, lists, blockquotes, tables, fenced code blocks, footnotes, and images.
