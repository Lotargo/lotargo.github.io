# Пример статьи

Это минимальный Markdown-источник для новой публикации. Первый абзац автоматически становится лидом статьи.

## Первый раздел

Опишите идею, аргументы и структуру. Рендерер сам создаст двуязычную HTML-страницу, строку метаданных, типографику, навигационную оболочку и компоненты изображений на основе Markdown и `article.json`.

> Markdown остаётся читаемым семантическим источником, а HTML становится сгенерированным артефактом публикации.

## Поддерживаемые элементы

Стандартный рендерер поддерживает заголовки, абзацы, выделение, ссылки, списки, цитаты, таблицы, fenced code blocks, сноски и изображения.
