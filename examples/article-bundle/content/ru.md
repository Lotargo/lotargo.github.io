# Пример статьи

Это минимальный Markdown-источник для новой публикации.

## Первый раздел

Сначала сформулируйте идею, аргументы и структуру. Затем подготовьте соответствующий статический HTML, используя общую дизайн-систему блога.

> Markdown остаётся читаемым источником, а HTML становится оформленной публикацией.
